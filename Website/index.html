HOW TO USE THE CONFESSIONS GENERATOR?


  1. Enter a phrase that you want to be included in the confession.
  2. Select required length of confession from the given radio buttons.
  3. Click on Generate Confession!
For example, input string : "I have a crush on"
and confession length : Short
